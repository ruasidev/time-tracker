# time tracker

A lightweight, frontend-first time tracking and weekly budgeting web app built with React, Vite, and TypeScript.

Designed to give a clear weekly view of how time is spent across projects, without accounts, servers, or unnecessary complexity. All data is stored locally for now.

## features

- weekly overview with actual vs target project hours
- workday entry tracking with notes and quarter-hour precision
- weekly report drawer with budget summaries and full entry timeline
- keyboard-friendly input and clean UI

## tech

- react
- vite
- typescript
- tailwind css
- localStorage

## development

npm install  
npm run dev

## notes

- no auth yet (planned)
- no backend yet (frontend-first by design)
- single local profile
